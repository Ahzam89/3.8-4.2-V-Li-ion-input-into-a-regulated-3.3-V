# 3.8-4.2-V-Li-ion-input-into-a-regulated-3.3-V
The board converts a single-cell 3.8–4.2 V Li-ion input into a regulated 3.3 V rail supporting up to 500 mA.  This was not simply a matter of placing an LDO between two connectors. The design required careful management of voltage headroom, protection losses, thermal dissipation, transient behavior, grounding, and output stability.
